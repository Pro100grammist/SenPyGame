Controls.

To move the character left and right, use the ⬅ ➡ keys
Jump ⬆
To double jump, press ⬆ again while the character is in the air
SPACE - melee attack or approval in the inventory menu, merchant, skill tree
E - ranged attack
Q - casting a spell
CTRL - change the spell scroll
SHIFT - change the elixir
F - use of the elixir
I - inventory
B - skill tree
C - character menu
X - interaction with objects on the map

To exit any menu, press the key of this menu again or ESC

UA

Робоча назва проект: Some Simple Game

Опис версії: комп'ютерна гра перебуває в альфа-версії. У грі ви керуєте персонажем і пробуєте пройти різноманітні рівні, збираючи бонуси, долаючи перешкоди та перемагаючи ворогів.
Гра спроектована в стлі двовимірного платформеру з елементами рольової гри. Так, по мірі проходженню гри, гравець буде розвивати свого персонажа за рахунок відкриття спеціальних
умінь та навчиок в обмін на очки досвіду, які він буде отримувати щоразу при підвищенні рівня. Також на характеристики гравця буде впивати зброя та екіпірування, яке можна знайти
в скринях, придбати у торговців або вибити з сиьного моба. Крім цього, в грі є безліч різноманітних корисних предметів, таких як елісири, книги магії та свитки заклинань, ключі від 
скринь з екіпіруванням та багато чого іншого.

Управління

Для переміщення персонажа вліво та вправо використовуйте клавіші ⬅ ➡
Стрибок ⬆
Для подвійного стрибка потрібно повторно прожати ⬆ доки персонаж в повітрі
SPACE - ближня атака або схвалення в меню інвентаря, торговця, дереві навичок
E - дальня атака
Q - кастування заклинання
CTRL - зміна свитка закляття
SHIFT - зміна еліксиру
F - використання еліксиру
I - інвентар
B - дерево навичок
С - меню персонажа
X - взаємодія з обєктами на карті

Щоб вийти з юудь-якого меню натисність ще раз клавішу цього меню або ESC
